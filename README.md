2x2 Project

🚀 Overview

2x2 is a decentralized cryptocurrency project designed for long-term sustainability, network security, and predictable coin issuance. The project uses a hybrid Proof-of-Work (PoW) and Proof-of-Stake (PoS) consensus mechanism with a fixed premine and a controlled annual coin emission schedule.

2x2 aims to provide:

Fair initial distribution with premine

Sustainable and predictable inflation

Long-term security incentives for miners and validators

Clear roadmap for coin supply and tail emission

💰 Coin Supply & Inflation

Premine: 11,000,000 coins

Annual coin emission: Varies by year, based on reward schedule

Blocks per year: 262,800

Tail emission (year 20+): 1,000 coins per block

⚙️ Consensus Mechanism
Proof-of-Work (PoW)

Rewards decrease over time according to a deterministic block schedule

Initial premine at genesis block: 11,000,000 coins

Designed to incentivize early miners while maintaining predictable supply

Proof-of-Stake (PoS)

Rewards are equal to PoW rewards per block after early phase
